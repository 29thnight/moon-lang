# Moon 언어 목표

## 비전

Moon는 Unity 게임플레이 스크립팅에 특화된 언어다. Kotlin에서 영감을 받은 간결한 문법으로 Unity 보일러플레이트를 제거하면서, C#으로 트랜스파일되어 Unity 파이프라인에서 투명하게 동작한다.

## 우선순위 (높은 순)

1. **간결한 Unity 게임플레이 작성** — 플레이어 이동, UI 연결, 트리거 처리, 코루틴 연출을 최소 코드로
2. **강력한 null safety** — 컴파일 타임 null 안전성으로 NRE 사전 방지
3. **Unity 친화적 의미론** — component, asset, lifecycle, serialize가 1급 언어 구성요소
4. **AOT 안전한 lowering** — IL2CPP 호환, 리플렉션/런타임 코드젠 없음
5. **디버깅 용이성** — 생성된 C#이 읽기 쉽고 Unity 디버거에서 직접 추적 가능
6. **점진적 구현 가능** — 최소 문법부터 시작해 안정적으로 확장

## 비목표

- 범용 프로그래밍 언어가 아님 (Unity 스크립팅 전용)
- 커스텀 런타임/VM 없음 (Unity의 Mono/IL2CPP 사용)
- Kotlin 호환이 아님 (Kotlin 영감을 받은 새 언어)
- 메타프로그래밍/매크로 없음
- C# 소스 인라인 불가 (intrinsic 블록으로 대체)

## 핵심 사용 사례

- 플레이어/컨트롤러 스크립트
- 트리거/충돌 로직
- UI 버튼/슬라이더 연결
- 코루틴 기반 타이밍 연출 (무적 시간, 페이드, 웨이브 스폰)
- 간단한 상태 기반 적 AI
- ScriptableObject 데이터 에셋
