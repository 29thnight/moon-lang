# Future Direction

The repository planning direction is clear even though not all of it is implemented yet.

Near-term next steps include:

- stronger editor tooling through a dedicated LSP server
- broader source-map driven debugging support
- wider integration-test coverage
- v2-style project graph and typed analysis expansion

This page is intentionally direction-setting, not a claim that those features already ship.
