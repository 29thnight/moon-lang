# Implementation Plan

현재 구현 계획은 마일스톤 기반으로 관리되고 있습니다.

완료된 마일스톤 범위:

- 언어 스캐폴드 및 샘플
- lexer
- parser
- semantic analysis
- lowering
- code generation
- CLI
- Unity package integration
- VS Code extension integration

현재 활성 마일스톤은 통합 테스트 확장으로, 더 많은 문법 커버와 negative case 보강이 중심입니다.
