# Mini Spec

현재 구현된 언어의 압축 요약은 대략 다음 범위를 중심으로 합니다.

- 파일당 하나의 최상위 선언
- `component`, `asset`, `class`, `data class`, `enum`, `attribute`
- null-safe type 과 generic collection reference
- 괄호 없는 제어문
- lifecycle block
- coroutine syntax 와 wait DSL
- `listen` event sugar
- `intrinsic` escape hatch
- `.prsmproject` 기반 프로젝트 범위

간단 스펙의 원본은 여전히 저장소 계획 문서에 있지만, 구현된 표면은 이 문서 사이트 전체에 반영되어 있습니다.
