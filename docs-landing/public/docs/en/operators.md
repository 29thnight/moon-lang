# Operators

## Arithmetic and comparison

- `+`, `-`, `*`, `/`, `%`
- `==`, `!=`, `<`, `>`, `<=`, `>=`

## Logic and assignment

- `&&`, `||`, `!`
- `=`, `+=`, `-=`, `*=`, `/=`, `%=`

## Null-safety and ranges

- `?.` safe access
- `?:` Elvis/null-coalescing
- `!!` non-null assertion
- `..` inclusive range
- `until`, `downTo`, `step` in loop-oriented range syntax
